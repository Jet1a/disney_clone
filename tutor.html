Transition & Transform

Transition เป็นคุณสมบัติใน CSS
ที่ช่วยให้สามารถเปลี่ยนค่าของคุณสมบัติ CSS ได้อย่างราบรื่นในช่วงเวลาที่ระบุ
ใช้สําหรับการสร้างภาพเคลื่อนไหวที่ราบรื่นโดยค่อยๆเปลี่ยนค่าของคุณสมบัติ CSS เมื่อเวลาผ่านไป

  • Property: ระบุคุณสมบัติ CSS ที่มีผลต่อการเปลี่ยนแปลง
  • Duration: ระบุระยะเวลา ในการเคลื่อนไหวเมื่อเวลาผ่านไปจนเสร็จ (e.g., 1s for 1 second).
  • Timing Function: กําหนดความเร่งของการเปลี่ยนแปลง (e.g., ease, linear, ease-in-out).
  • Delay: คุณสมบัติเสริมเพื่อเพิ่มความล่าช้าก่อนที่การเปลี่ยนแปลงจะเริ่มขึ้น

nth-child Select

คือ pseudo-class ที่ใช้เพื่อเลือกและจัดรูปแบบองค์ประกอบตามตําแหน่งภายในองค์ประกอบหลัก ช่วยให้สามารถกําหนดเป้าหมายองค์ประกอบที่ต้องการแก้ไข เฉพาะ เจาะจงได้ ให้ประสิทธิภาพในการนําสไตล์ไปใช้กับองค์ประกอบเฉพาะในลักษณะที่ยืดหยุ่นตามตําแหน่งในโครงสร้างเอกสาร
  • :nth-child(odd) selects all odd-numbered child elements.
  • :nth-child(even) selects all even-numbered child elements.
  • :nth-child(3n) selects every third child element.

Object Fit

คุณสมบัติ CSS ที่ใช้สำหรับปรับขนาดองค์ประกอบของ < img > หรือ < video > มีประโยชน์เมื่ออัตราส่วนกว้างยาวของเนื้อหาและอัตราส่วนกว้างยาวของคอนเทนเนอร์ไม่ตรงกัน
  • fill เนื้อหาจะถูกปรับขนาดให้เต็มคอนเทนเนอร์โดยสมบูรณ์ ซึ่งอัตราส่วนภาพอาจไม่พอดี
  • contain เนื้อหาจะถูกปรับขนาดให้พอดีกับคอนเทนเนอร์ในขณะที่ยังคงอัตราส่วนภาพไว้ แต่อาจไม่ครอบคลุมทั้งคอนเทนเนอร์ทําให้มีพื้นที่ว่าง
  • cover เนื้อหาจะถูกปรับขนาดให้ครอบคลุมคอนเทนเนอร์ทั้งหมดในขณะที่ยังคงอัตราส่วนภาพไว้ ซึ่งอาจส่งผลให้บางส่วนของเนื้อหาถูกครอบตัด
  • none (default) เนื้อหายังคงขนาดเดิมโดยไม่ต้องปรับขนาดใด ๆ อาจล้นคอนเทนเนอร์หรือปล่อยให้พื้นที่ว่างขึ้นอยู่กับอัตราส่วนภาพ
  • scale-down เนื้อหาจะถูกปรับขนาดให้เล็กลงตามขนาดที่จริงเพื่อให้พอดีกับคอนเทนเนอร์บรรจุ จะไม่ปรับขนาดหากไม่จําเป็น
